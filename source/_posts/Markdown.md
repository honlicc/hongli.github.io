## 标题 \#
###  通过 \# 表示，只需要在文本前面加上 # 即可
```
#  标题1
##   标题2
###    标题3
####    标题4
#####    标题5
######    标题6
```
## 字体
### 加粗、代码高亮、斜体、删除线
```
//斜体加粗
***斜体加粗***
//加粗  
**加粗**
//代码高亮
==代码高亮==
//斜体
*斜体*
//删除线
~~被删除的文字~~
```
#### 效果展示
***斜体加粗***
**加粗**
*斜体内容*
==代码高亮==
~~被删除的文字~~
